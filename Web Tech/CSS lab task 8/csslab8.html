Overflow Handling Examples

1. Overflow: visible

This is an example of overflow set to visible. The content will spill outside the box if it exceeds the fixed size.

2. Overflow: hidden

This is an example of overflow set to hidden. Any content that exceeds the box will be clipped and not visible.

3. Overflow: scroll

This is an example of overflow set to scroll. Scrollbars will always appear, allowing you to scroll through content that exceeds the box size.

4. Overflow: auto

This is an example of overflow set to auto. Scrollbars will appear only if the content exceeds the box size.
